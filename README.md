# Biblioteka-Projekt
Boże co my robimy

### Wymagania
- Python 3.12 or higher
- Nodejs v22.14.0

## Instalacja
### Back-End (Python/Flask)
```
cd flask-server
python -m venv {nazwa środowkiska}
./{środowisko}/Script/activate
pip install flask
python server.py```
```
### Front-End (React)
```
cd client 
npm install 
npm start
```
